Meet FedoraCoin.

FedoraCoin is an easy to use, optionally anonymous, coin with value and utility.

Users of the currency can thank each other for great contributions in online communities by tipping some fedoras to each other.

Having issues connecting and syncing with the network?

Add the following lines to your FedoraCoin.conf file (should be at C:\Users\[username]\AppData\Roaming\FedoraCoin\FedoraCoin.conf, or ~/.fedoracoin/fedoracoin.conf):

addnode=coins.prohashing.com:6150 
addnode=45.55.244.201:44890 
addnode=[2001:0:9d38:90d7:c30:20c3:e886:df00]:44890
